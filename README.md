# Projeto-final
ESTAÇÂO METEREOLÓGICA COM FreeRTOS ESP32
ESP32 com FreeRTOS, DHT22 e OLED SSD1306.

Para esse projeto, foi utilizado de recursos de gerenciamento de tarefas de um sistema operacional para realizar medidas de um estação metereológica, equipada com o sensor de temperatura e umidade DHT 22, um medidor de velocidade do vento, simulado por um potenciômetro e um display OLED. Para cada uma dessas tarefas foi criada uma task, com diferentes prioridades.
